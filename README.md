# Neuronomator

[![Build Status](https://travis-ci.org/lonski/neuronomator.svg?branch=master)](https://travis-ci.org/lonski/neuronomator)

Neural network backpropagation alghoritm implementation.

## Building

```
mkdir build && cd build && cmake .. && make
```

## Running

### Main

```
./neuralnet
```

### Tests

```
./neuralnet_test
```
